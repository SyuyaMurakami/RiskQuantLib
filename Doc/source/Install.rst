Install
============

.. toctree::
   :maxdepth: 4

To install RiskQuantLib, you can use ``pip`` command in terminal as follows:
::

   pip install --index-url https://test.pypi.org/simple/ --no-deps Syuya-Murakami --upgrade

RiskQuantLib can also be downloaded in Github: `RiskQuantLib <https://github.com/SyuyaMurakami/RiskQuantLib>`_
