buildInstrument(filePath, targetProjectPath = ‘’) is a function to automatically build instrument classes. If instrument already exists, it will skip this instrument. Any instrument that doesn’t exist in excel file won’t be built and added into RiskQuantLib path, regardless of whether it used to exist in target project.

filePathstr

The excel file path that tells RiskQuantLib how to build instruments, including the name of instruments, which instrument class it inherits from, which QuantLib class it inherits from, and instrument default type string.

targetProjectPathstr

The RiskQuantLib project path where you want to build instruments. You can leave this parameter empty to build instruments in this project. Or specify a path to build instruments to another RiskQuantLib project.
